# E2E Character Test Suite Catalog

Authoritative index of every comprehensive character build spec under
`test/e2e/specs/tgtt-*.spec.ts`.  Each spec is generated by the
`describeCharacter` factory and runs the canonical 7-test set
(L1 / L3 / L5 / L5-loadout / MEGA L1→20 / USE probes / L1 export
round-trip) plus the multiclass plan for multiclass entries.

For the full *what* and *why* of every check, see the
[Comprehensive Test Standard](./comprehensive-test-standard.md) and
the skill reference at
[`.agents/skills/e2e-character-tests/references/standard.md`](../../.agents/skills/e2e-character-tests/references/standard.md).

## How to add a new entry

1. Author the new spec following
   [`spec-template.md`](../../.agents/skills/e2e-character-tests/references/spec-template.md).
2. Run the canonical 7-test set locally (see
   [Comprehensive Test Standard](./comprehensive-test-standard.md)).
3. Add a row to the table above with build, level scope, and the
   single-line "what it proves" hook.
4. Update
   [`test/e2e/specs/README.md`](../../test/e2e/specs/README.md) to
   match.
